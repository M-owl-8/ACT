═══════════════════════════════════════════════════════════════════════
  METHOD 1: INSTALL VIA USB (RECOMMENDED)
═══════════════════════════════════════════════════════════════════════

1. ENABLE USB DEBUGGING (if not already done):
   - Go to Settings → About Phone
   - Tap "Build Number" 7 times (Developer Mode enabled)
   - Go back → Developer Options
   - Enable "USB Debugging"

2. CONNECT YOUR PHONE:
   - Connect phone to laptop via USB cable
   - Allow USB debugging when prompted on phone

3. INSTALL APK:
   Open PowerShell and run:
   
   adb install "C:\Users\user\Desktop\ACT-Gen1-Standalone.apk"

   OR simply run:
   
   adb install -r "C:\Users\user\Desktop\ACT-Gen1-Standalone.apk"
   
   (The -r flag replaces existing app if already installed)

4. DONE! 
   - Look for "ACT Gen-1" app on your phone
   - Open it and create an account
   - Disconnect phone from laptop
   - App works completely offline! ✅

  METHOD 2: TRANSFER APK TO PHONE (NO USB DEBUGGING)
═══════════════════════════════════════════════════════════════════════

1. TRANSFER APK TO PHONE:
   Choose one method:
   
   a) Via USB Cable:

      - Copy "ACT-Gen1-Standalone.apk" to phone's Downloads folder

═══════════════════════════════════════════════════════════════════════
  🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

PROBLEM: "App not installed" error
SOLUTION: 
  - Uninstall old version first (if exists)
  - Enable "Install unknown apps" in Settings
  - Try installing again

PROBLEM: "Parse error"
SOLUTION:
  - APK file might be corrupted during transfer
  - Re-copy the APK from laptop to phone
  - Make sure file size is 115 MB

PROBLEM: App crashes on startup
SOLUTION:
  - Clear app data: Settings → Apps → ACT Gen-1 → Storage → Clear Data
  - Restart phone
  - Open app again

PROBLEM: Can't create account
SOLUTION:
  - Make sure password is at least 6 characters
  - Make sure passwords match
  - Try a different email
